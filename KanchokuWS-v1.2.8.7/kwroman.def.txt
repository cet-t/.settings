#
# 漢直WS ローマ字定義ファイル
#
#   - このファイルは起動時と再読込時にロードされる。
#
# 形式: ローマ字パターン <空白> カタカナ文字列
#
# 変換規則:
#   - ローマ字部分は英文字で記述する(大文字/小文字は区別しない)
#   - ^$%@[] 以外の記号をローマ字の一部として使うこともできる
#   - [・・・] はローマ字前後の制約を定義する文字クラスを表す
#     - 例: M[BMP] : M の後に B, M, P が後接する場合にマッチ
#   - [^・・・] ば、文字クラスに含まれない場合にマッチ
#   - $ は文字列の先頭または末尾にマッチ
#   - @ は母音(AIUEO)にマッチ
#   - % は子音にマッチ
#   - ローマ字パターンの末尾が子音の場合は、その直後も子音または語末であると想定する
#     - 例: 「BR  バー」は「BR[%$]  バー」と同等
#   - ^$%@[] とローマ字の一部として使われている記号以外の記号はローマ字変換のストッパーになる
#   - 
#
# 処理順:
#   - ローマ字部分が長い方を先に処理
#   - ローマ字部分が同じ長さなら記述順に処理するので、より詳細な定義を先に書くこと
#
# 例:
#  - [%$]@X$    ックス
#      "X" が末尾にきて、その直前が母音であり、2文字前が子音または語頭であるなら "X" を「ックス」に変換。
#      "Sax" ⇒ 「サックス」
#      "Ox"  ⇒ 「オックス」
#  - M[BMP]     ン
#      "M" の直後が B,M,P のいずれかなら "M" を 「ン」に変換。
#      "Pomp"  ⇒ 「ポンプ」
#      "Gamma" ⇒ 「ガンマ」
